# @shined/babel-plugin-enforce-direct-access

A Babel plugin to enforce direct access patterns.

## Installation

```bash
npm install @shined/babel-plugin-enforce-direct-access
```

## Usage

```javascript
// babel.config.js
module.exports = {
  plugins: [
    ['@shined/babel-plugin-enforce-direct-access', {
      // options
    }]
  ]
};
```
