# @shined/swc-plugin-enforce-direct-access

SWC plugin for enforcing direct access patterns.

## Installation

```bash
npm install @shined/swc-plugin-enforce-direct-access
```

## Usage

```javascript
// .swcrc
{
  "jsc": {
    "experimental": {
      "plugins": [
        ["@shined/swc-plugin-enforce-direct-access", {
          // options
        }]
      ]
    }
  }
}
```
